- Ilja van Sprundel (https://github.com/iljavs)
- Michael Smith (https://github.com/michaelshmitty)

## Setup and run FreeBSD VM

See [VM setup instructions](https://github.com/iljavs/FreeBSD-Jail-Security-Research/blob/main/vm/vm-setup-instructions.md).

Run VM with:

### On amd64 host

```sh
cd vm
./run.sh
```

### On Apple Silicon

```sh
cd vm
./run_apple.sh
```

## Run online FreeBSD kernel debugger VM

See [Online debugger instructions](https://github.com/iljavs/FreeBSD-Jail-Security-Research/blob/main/vm/debugger-instructions.md).

Run debugger vm with:

### On amd64 host

```sh
cd vm
./run_debugger.sh
```

### On Apple Silicon

```sh
cd vm
./run_debugger_apple.sh
```
